Prerequisites

The following hardware and software is required to run eBlitz Suite:

Hardware
Hardware requirements vary depending on sizing and deployment type.
For a distributed, production environment, the requirements are:

• For each physical node, an Intel Pentium Dual Core processor, 2 GB RAM, 1 GB free disk space and a network card that supports 1 Gb/sec.
• Network bandwidth of 1 Gb/sec or higher.

For development and evaluation purposes,eBlitz Suite can run on a single machine, even on a notebook.

The hardware requirements in this case are:

• Intel Pentium IV processor, 1 GB RAM, 1-2 GB free disk space and a network card that supports 100 Mb/sec.

• Network bandwidth of 100Mb/sec or higher.

Software

Each physical node must have the following software:

• Java SE Development Kit 6u10 (You can download this from http://java.sun.com/javase/downloads/index.jsp)
• JAVA_HOME environment variable set to point to the Java JDK.
• PATH and CLASSPATH environment variable set to point to the Java JDK Bin and JRE Bin.

Supported Platforms

eBlitz supports the following Operating System and browsers:

• Microsoft Windows 2003 Server
• Microsoft Internet Explorer 6 and 7
• Mozilla Firefox 2

Basic Installation

The GUI installer is the easiest way to install eBlitz suite. However, it may not be suitable for all the configuration options required by production environments.

Production installation and deployment is described in detail in Chapter 4, Deploying eBlitz Suite

This chapter contains the following sections:
• Using the GUI Installer
• After GUI Installation

Using the GUI Installer

Before using the GUI Installer make sure that you have a correctly setup environment.
For hardware and software requirements, and supported platforms, see Chapter 1, Getting Started.
For an evaluation environment, you need valid credentials to a configured database. For details, see Preparing Database Section.

JBoss does not require any additional configuration for evaluation purposes. If you are using the GUI installation for a production environment with JBoss or for a different application server, see Setting Up Application Servers Section.

 

To install using the Installation Wizard:

  1. Make sure the JBOSS Application Server or any other server is not running.
  2. Do one of the following:
    1. Execute the file eBlitzSuite .jar, located on the installation CD or in your distribution directory.
    2. Execute the following command:

java -jar eBlitzSuite.jar
           
            The Installation Wizard opens displaying the Welcome page.

  1. In the Welcome page, eBlitz line of Products are shown. Click on Next. eBlitz Suite Description page opens.
  2. In eBlitz Suite Description Page, review the hardware and software requirements, and then click Next.
  3. In the License page review the license, select I Accept the Terms of the License Agreement and then click Next. Servlet Container page opens.
  4. In the servlet container page, default selected servlet container is Apache tomcat and click Next.
  5. Servlet container installation folder location page opens. The default location of apache-tomcat-6.0.18 servlet container installation folder is: C:/eBlitz/apache tomcat. If there is any such folder exists in the same location, please move the folder to some other location. Click on Next.
  6. In the Application Server page, default selected application server is JBOSS-4.2.2 and click Next.
  7. Application Server installation folder location page opens. The default location of JBOSS-4.2.2 Application Server installation folder is: C:/eBlitz/jboss-4.2.2.GA. If there is any such folder exists in the same location, please move the folder to some other location. Click on Next.
  8. In the ESB page, default selected ESB is eBlitz ESB and click Next.
  9. In the Governance page, default selected governance tool is eBlitz Governance and click Next.
  10. In the BPEL Engine page, default selected BPEL tool is eBlitz BPEL Engine and click Next.
  11. The installer validates the data image. The installer verifies the data and the eBlitzSuite Platform V1.0 installation process starts.
  12. In the Installation Progress page, track each step of the installation. When the installation is complete, click Finish to exit the installation wizard.

4 After the Installation

  • To start the eBlitz Suite ,go to the All Programs menu of the system ,under eBlitz click on start eBlitz suite.
  • This starts the application server which is JBoss by default.Then click on the start console in the eBlitz Suite.
  • The console would be launched and then the BPEL ,ESB and Governance tools could be initiated from the START button on the console.
  • Also links have been provided to navigate to the console of the respective tools .
  • The STOP button would stop the underlying tool.
