INSTALLATION

1. Setup an application context on your application server to host a web application. For assistance with setting up a web application, refer to this knowledge base article:  

http://support.businessobjects.com/library/kbase/articles/c2016911.asp

2. Extract the contents of this zip file into a your web application's context root.

3. In a browser window, launch the file create_report_from_scratch.jsp as follows: 

http://<computer name>:<port number>/<virtual directory>/create_report_from_scratch.jsp
