digiBlitz ESB is an ESB (Enterprise Service Bus) that combines the functionality of a Service Oriented Architecture (SOA) and an Event Driven Architecture (EDA) to create an agile, enterprise ESB.

The Enterprise Service Bus (ESB)- which can be defined as middleware that brings together both integration technologies and runtime services to make business services widely available for reuse - offers the best solution for meeting today's enterprise application integration challenges by providing a software infrastructure that enables SOA.

Prerequisites :

  • Java Developer Kit (JDK) 1.5.x (Java 5). For downloading and installation, please refer to: http://java.sun.com/javase/downloads/
  • Apache Maven 2.x. For downloading and installation, please refer to: http://maven.apache.org/users/index.html
  • Apache Servicemix-3.2.3. For downloading, please refer to: http://servicemix.apache.org/download.html & extract the download into a directory of your choice. Add $SERVICEMIX_HOME directory to $SERVICEMIX_HOME environment variable and add $SERVICEMIX_HOME/bin directory to $PATH environment variable.
  • Graphviz - Graph Visualization Software for component and endpoint visualisation. For downloading and installation, please refer to: http://www.graphviz.org/Download.php
